Задача: Система управления врагами (Wave Manager)

Суть задачи:

 Тебе нужно создать программу, которая симулирует появление врагов и их уничтожение игроком. 
 Главное условие — использовать List<Enemy> для хранения списка врагов, 
 так как их количество постоянно меняется.

Требования к реализации (Что нужно сделать):
1. Создать класс Enemy:
   - У него должны быть поля (или свойства): string Name, int Health, int Damage.
   - Добавь конструктор для удобного создания врага 
     (например, public Enemy(string name, int health, int damage)).

2. Главное меню (цикл программы):
   - При запуске программа показывает меню:
	1. Добавить врага
	2. Атаковать всех врагов
	3. Показать список врагов
	4. Удалить мёртвых врагов
	5. Выход
   - Программа должна работать в бесконечном цикле while (true), пока пользователь не выберет "Выход".
   - Используй switch для обработки выбора пользователя.

3. Логика пунктов меню (используем List):
   1. Добавить врага:
      - Спрашиваешь у пользователя имя, здоровье и урон.
      - Создаешь новый объект Enemy.
      - Добавляешь его в List<Enemy> 
         (список должен быть объявлен в начале программы, например, 
         static List<Enemy> enemies = new List<Enemy>();).
   2. Атаковать всех врагов:
      - Проходишь по ВСЕМ врагам в списке циклом for или foreach.
      - Каждому врагу наносишь урон (например, фиксированный урон от игрока, скажем, 20 HP).
      - Выводишь сообщение: "Игрок атакует [ИмяВрага]! У врага осталось [Health] HP".
      - Важно: Атака не должна удалять врага из списа здесь и сейчас (это может сломать цикл foreach). 
         Мы только меняем их здоровье.
   3. Показать список врагов:
      - Проходишь по списку циклом for и выводишь информацию о каждом живущем враге: 
         "Враг №1: Орк (50 HP, 10 урона)".
      - Если список пуст, написать "Врагов нет".
   4. Удалить мёртвых врагов:
      - Здесь тебе нужно пройти по списку в обратном порядке 
         (или использовать for (int i = list.Count - 1; i >= 0; i--)), 
         чтобы найти врагов, у которых Health <= 0.
      - Удалить их из списка с помощью enemies.RemoveAt(i).
      - Вывести сообщение: "Мёртвые враги удалены с поля боя.".

Дополнительные условия (прокачка скилла):

- Random: 
   При создании врага (пункт 1) дай возможность пользователю ввести random вместо имени, 
   чтобы сгенерировать случайного врага из заранее заготовленного массива строк 
   {"Гоблин", "Орк", "Скелет", "Демон"}, а здоровье и урон тоже сделать случайными 
   (например, Health от 30 до 80, Damage от 5 до 20). Используй класс Random.

- ООП: 
   Вынеси всю логику работы со списком (добавление, удаление мертвых, вывод) 
   в отдельный статический класс EnemyManager, чтобы основной код программы (меню) был чистым.

- LINQ: 
   В пункте "Показать список врагов" попробуй показать ТОП-3 самых сильных врагов по урону, 
   используя OrderByDescending и Take.
